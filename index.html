如何启用本地模型（模式 B）？

  1. 确保已安装 Ollama

    前往 ollama.com 下载并安装。

    打开终端（CMD 或 PowerShell）运行：ollama pull deepseek-r1:7b

  2. 启动 Ollama 并允许网页访问

    Windows：

    set OLLAMA_ORIGINS=*
ollama serve

    macOS / Linux：

    export OLLAMA_ORIGINS=*
ollama serve
  3. 在设置中输入 Ollama 地址

    默认地址：http://localhost:11434

  4. 点击“测试 Ollama 连接”并选择模型
